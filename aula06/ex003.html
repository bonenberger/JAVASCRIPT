comandos e variáveis

exemplos de comandos:

  • window.alert: mostra uma mensagem na tela
  • window.prompt: abre um campo para o usuário digitar

variáveis

para conversão de variáveis também temos alguns comandos

  • Number.parsefloat(n): converte um tipo em tipo Number float
  • Number.parseInt(n): converte um tipo para tipo Number inteiro
  • Number(n)
  • String(n): converte um tipo em string
  • n.toString()
  • n.lenght: informa o tamanho da string
  • n.toUpperCase: transforma a strig toda em letras maiúsculas
  • n.toLowerCase: transforma a string toda em letras minúsculas

template string

nas versões mais atuais do JS foi implementada a tecnologia de template string, que consiste em colocar a string entre crases (``) e utilizar o placeholder ${var} para concatenação.

funciona assim: você cria as variáveis:

  • var nome = 'rafael'
  • var idade = 45
  • var nota = 9.5

se você quiser criar uma string concatenada com essas informações basta fazer como segue: `o aluno ${nome} de ${idade} conseguiu a nota ${nota} no exame final` o que gera a frase completa
