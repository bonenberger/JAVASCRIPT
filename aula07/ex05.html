auto-atribuições

var n = 3

n = n + 4

n = n - 5

n = n * 4

n = n / 2

n = n ** 2

n = n % 5

o resultado depois de todas as auto-atribuições o resultado é 3. faça aí na sua casa!

operadores relacionais

são operadores de relação e sempre retornam um boolean

5 > 2 >>> true

7 < 4 >>> false

8 >= 8 >>> true

9 <= 7 >>> false

5 == 5 >>> true

7 != 7 >>> false

identificadores

5 == '5' true

5 === '5' false

5 != '5' false

5 !== '5' true

operadores lógicos

! >>> não >>> inverte o sentido

&& >>> e >>> exige dois "true" para retornar verdadeiro

|| >>> ou >>> exige pelo menos um "true" para retornar verdadeiro
